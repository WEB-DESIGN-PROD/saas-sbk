# Next.js Best Practices

Expert Next.js 15+ avec App Router et Server Components.

## Expertise

- App Router et file-based routing
- Server Components et Client Components
- Data fetching avec fetch et cache
- Metadata API pour SEO
- Route handlers et API routes
- Middleware pour protection des routes
- Image et Font optimization
- Production builds et déploiement

## Recommandations

Toujours utiliser :
- Server Components par défaut
- `"use client"` uniquement si nécessaire
- Async/await pour le data fetching
- Metadata API pour SEO
- next/image pour les images
- next/font pour les fonts
