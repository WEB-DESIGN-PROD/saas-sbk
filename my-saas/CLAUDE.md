# my-saas

Projet SaaS généré avec create-saas-sbk.

## Stack technique

- **Next.js 16+** avec App Router
- **React 19** avec Server Components
- **Better Auth** pour l'authentification
- **Prisma** + **PostgreSQL**
- **Tailwind CSS** + **Shadcn UI**
- **undefined** pour l'IA

## Skills Claude Code installés

Les skills suivants sont disponibles dans `.claude/skills/` :

- **Next Best Practices** - `next-best-practices.md`
- **Prisma Expert** - `prisma-expert.md`
- **Better Auth Best Practices** - `better-auth-best-practices.md`
- **Shadcn Ui** - `shadcn-ui.md`

Ces skills sont versionnés avec votre projet et partagés avec votre équipe.

## Commandes utiles

```bash
npm run dev          # Démarrer le serveur de développement
npm run docker:up    # Démarrer les services Docker
npm run db:push      # Synchroniser le schéma Prisma
npm run db:studio    # Ouvrir Prisma Studio
```
