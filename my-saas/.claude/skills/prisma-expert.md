# Prisma Expert

Expert Prisma ORM pour Next.js avec PostgreSQL.

## Expertise

- Schéma Prisma et modèles
- Migrations de base de données
- Prisma Client et requêtes

- Toujours utiliser `prisma generate` après modification du schéma
- Utiliser `prisma db push` en développement
- Utiliser `prisma migrate` en production
- Client Prisma en singleton pattern
- Typage TypeScript automatique
